FAQ's

General questions

What is this program?

This program trains you for a New Economic Order, creating on open source products while working on two different polarities such as traditional finance and alternative finance. Thus mixing both.

What is the requirement to join this program?

Anyone can join this program as long as there is nothing holding you back.

Is this a paid program?

This is completely free program for students. We encourage you and others to donate time/money for mentors.

What kind of skills are required to join this program?

You need to be able to learn how to use a computer.You should have some soft skills with an eagerness to learn and apply that knowledge for building social sustainable businesses.
Mentors

What is in for you?

Build your tribe, share what you have learned to the new and build a talent pool around you..

Do you want to get paid?

As a mentor, if you wish to get paid: As of now, we are not paying to mentors. Although you can ask students for direct payments but only in such cases:
  • If a student wants extra time.
  • It's better if you ask students to contribute and help in your open source projects
  • Run a crowdfunding campaign and donate some to Learn program.

What kind of projects you can teach?

So, you can teach , anything which can be hosted on github, run as a small community, make students learn and grow , allow students to work from remote, data science, marketing , design anything etc.

Why mentor here?

There are a number of reasons why you should join us:
  • This is a global program so, you can imagine impact.
  • There is current talent deficit and skill gap in working methodologies and how our industries have been operating so far. We wish to bring that change in attitude with this community.
  • You get to train students online on projects, which benefit everyone and create a pool of talent which you can use on your further endeavours, experiments or existing project.
